# Demo
Това е примерно задание

**Този проект представя алтернативна цветова схема за типовете земно покритие според CORINE Land Cover, вдъхновена от картината „Целувката“ на Густав Климт.**

 От избраната картина е извлечена цветова палитра чрез анализ на доминиращите тонове и цветови акценти. След това палитрата е приложена към всички категории земно покритие в CORINE.
